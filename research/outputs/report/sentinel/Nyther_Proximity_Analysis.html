The single most important statement when asked about competition

Every competitor in this space solves the network layer or the physics layer — none does both, and none has ever addressed the thermodynamic layer. Liu et al. (2011) proved mathematically that any system confined to the network layer cannot detect the FDI attack class. Nyther Sentinel is the first system to operate across all three physical domains simultaneously — cyber, electrical physics, and thermodynamic — in a real-time, deployable, Indian-market-calibrated product. The closest competitor on physics (Argonne GPS) is a batch research tool using a linearised method that its own literature proves is bypassable. The closest competitors on network (Dragos, Claroty) produce zero FDI alerts by mathematical necessity. The academic literature provides the theoretical foundation — Nyther Sentinel provides the first working implementation that goes beyond it.
